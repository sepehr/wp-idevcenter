iDevCenter.com integration for WordPress.

== Description ==
Provides [idc-vote] shortcode which wnable you to show your post's iDC rank to the visitors.

== Installation ==
1. Upload `idevcenter.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu.
1. Place the `[idc-vote]` shortcode in a blog post that you wish to show its iDC rank.

== Frequently Asked Questions ==
= What if my post link has not yet been submitted to iDC? =
The button link directs your visitor to iDC submit form to submit the link.

= Does it add the iDC button to all of my blog posts? =
Ofcourse not! the iDC button will be added to the posts in which you dropped the `[idc-vote]` shortcode.
